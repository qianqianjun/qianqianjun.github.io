# 相识 #

回想起我们相识的过程，即便到现在，我依旧觉得是那么的不可思议。若不是我们同时勇敢了一次，可能就不会有这么美好的邂逅。或许这就是冥冥之中的缘分，或许我们就是彼此的命中注定！




咱俩真的太像了~



对对！



遇到雪雪，我第一次知道相见恨晚是什么感觉：雪雪不仅学习优秀，还是很负责任的学生干部，也是保研选手，宝藏女孩儿无疑啦，真的让我眼前一亮。在与雪雪聊天时，我能感觉到雪雪是那种责任心强，很有担当，乐观积极，并且有爱心的女孩子。更难得的是，我们的三观真的好合呀，并且性格也都是那种热情周到，温柔体贴的类型（这是那天咱俩测试性格发现的惊喜）。虽然和你相处的日子不多，但我总会有认识你好久的错觉。





第一次见面
















还记得第一次见面前，我被室友吓唬住了，很怕见面的时候尬住。雪雪知道后非常温柔的安慰我，甚至还提出了解决办法，哈哈哈，真有你的。当时真的被暖到了，心想怎么会有这么温柔的女孩子，也就不紧张了。第一次见面印象最深刻的就是咱俩吃完晚饭后围着地铁站散步，周围是熙熙攘攘的人群，和煦的微风吹拂，当我望向你眼睛的时候，仿佛看到了美丽的星星，雪雪的眼神是那么温柔，清澈见底~你的微笑也让我不由地心跳加速。雪雪那天对我说的那些话让我很感动，尤其是讲到你非常信任我，真的有点受宠若惊，我当时就暗下决心，绝对不能辜负这份信任！雪雪好像总能发现我的优点，并且从不吝啬自己的夸奖，每一次被你夸都能让我无比开心。有的时候雪雪还会化身为我的人生导师鼓励我，繁忙的科研生活偶尔真的会让人筋疲力尽，但你的鼓励总能让我重新燃起热情，谢谢你！


第二次见面



第一次见面之后，我每天仿佛就有了牵挂，会忍不住想找你聊天。约好见面的时间后，我第一次体会到了度日如年的感觉，每天都在盼望早点到约定的日子。见面前，我的心情是期待中夹杂着一些紧张，不过还好见面的时候，我们都打开了心扉，和对方分享了自己不堪回首的往事，也说出了各自的顾虑。很幸运的是咱俩的顾虑都不是什么大问题，可以一起努力克服。其实在那一刻，我就已经想好了。晚饭后我们在道科门口畅快的聊天，可能是觉得我俩聊得很投机，也可能是胳膊被主动挽了一下，我情不自禁的拉起了你的手，后来都不知道该咋收场了，哈哈哈哈。我当时害怕极了，怕给你留下不好的印象，怕以后没有机会再次牵起你的手（不过还好你没生气，谢天谢地）。晚上我认真的反思，越想越觉得咱俩挺合适的，所以，我那天也不算是冲动吧，毕竟我确实也想好了不是嘛。




第三次见面


我要非常感谢学校的运动会，不然那周就见不到雪雪啦，感恩感恩（虽然运动会害的雪雪5点半就要起床，哈哈哈哈）！在帮雪雪抢疫苗的时候，我其实很担心抢不到，甚至在想如果抢不到要怎么安慰你。不过那天真的运气爆棚，我们抢到了疫苗，说实话，我比你还开心，我激动的又拉起了你的手。可能你在帮闺蜜抢疫苗没反应过来，没有挣脱我，嘿嘿嘿~我拉着你的手静静的在旁边看着你，感觉好幸福，脑子里已经在想一会要带你去哪，说些什么，怎么表白了（哈哈哈，感觉气氛到了，想抓住机会）。不过后面我被问了一些让我摸不着头脑的问题，然后雪雪还一直跟我说要我都想好了……我当时有点乱，错把雪雪的好意理解成了拒绝，所以非常失落。虽然后面误会解除了，但送你去地铁站的路上咱俩又双叒叕谈起了深刻的人生问题，哈哈哈哈，我真的尽力了，没找到机会（手动捂脸）。



记得你问我次数最多的问题是我为啥会对你有好感（大概是这个意思叭~），也许是第一次见面时，你望向我那温柔的目光，清澈的让我着迷；也许是每天联系你时，和你相谈甚欢时的感觉，仿佛是相见恨晚；也许是想到我的未来能和你相伴时，抑制不住的那种期待与憧憬。但其实正确的答案很简单，我喜欢，因为是你，因为是你，我喜欢！


我们彼此信任，相互欣赏，都会为对方考虑，也会鼓励对方。如果能在一起的话，我们一定都会成为更好的自己！









有人说养成一个习惯需要21天的时间，偷偷告诉你，从我们相识的那天算起，到今天这个充满美好与祝福的日子，我已经养成喜欢你的习惯啦。多出来的6天，以及我们在一起之后的每一天，我都会多喜欢你一点点，直到永远。我想说，虽然你的过去我来不及参与，但你的未来，我想陪你一起走过！


我很想成为你的人生战友，与你荣辱与共同进退，一起携手向未来，陪伴着你走过漫长的岁月。亲爱的雪雪，做我女朋友好嘛！
